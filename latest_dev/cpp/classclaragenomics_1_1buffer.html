Detailed Description

template<typename T, typename Allocator>
class claragenomics::buffer< T, Allocator >

Container for an array of elements of type T in host or device memory. T must be trivially copyable.

This method reserves memory for the specified number of elements. If new_capacity is less than or equal to _capacity, this call has no effect. Otherwise, this method is a request for allocation of additional memory. If the request is successful, then _capacity is equal to the new_capacity; otherwise, _capacity is unchanged. In either case, size() is unchanged.

Parameters
new_capacityThe number of elements to reserve.
streamThe CUDA stream to be associated with this method.
