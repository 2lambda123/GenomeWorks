genomeworks.io.utils.read_poa_group_file(file_path, num_windows=0)¶

Parses data file containing POA groups.

Args:
file_pathPath to POA group file.

File format is as follows - <num sequences> seq 1… seq 2… <num sequences> seq 1… seq 2… seq 3…

num_windowsNumber of windows to extract from

file. If requested is more than available in file, windows are repoeated in a circular loop like fashion. 0 (default) implies using only those windows available in file.
